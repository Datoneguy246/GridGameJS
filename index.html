GridGame.js is an online, 2D, JavaScript game engine.

The engine runs on an 8 by 8 pixel grid
and the exported games can be played in a browser.




Documentation


Basic:

Log

Logs messages in the console

To Log:

Log("Message");

In built games, messages are logged in the browser console.

Objects

An object is a class representing a rectangle that can be drawn on the screen.

To create an object:

var newObject = new Object(x, y, width, height);

The x and y coordinates of an object are relative to the top-left corner of itself.

To assign an objects color:

newObject.color = myColor;

To draw an object:

function Update(){
    DrawObject(newObject);
}


Color

A color is a class representing... a color.

To create a color:

var newColor = new Color(r, g, b);

Alpha values are not yet supported. :(

To assign an objects color:

newObject.color = newColor;

To assign the color of the background:

SetBackground(newColor);


Input

All the functions used to obtain user input

To get keyboard input:

if(isPressing('z'))
{
    Log("Z key is pressed");
}

Currently, isPressing() only supports alpha-numeric characters.
