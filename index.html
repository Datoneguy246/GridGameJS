GridGame.js is an online, 2D, JavaScript game engine.

The engine runs on an 8 by 8 pixel grid
and the exported games can be played in a browser.




Documentation


Log

Logs messages in the console

To Log:

Log("Message");

In built games, messages are logged in the browser console.

Update Function

A built-in function that gets called every frame.

Usage:

function Update(){
    //Code goes here
}

After each frame, the screen will clear everything (except the background) so it is neccesary to draw things in the Update function in order for them to stay drawn.

Drawables

An drawable is a class representing a rectangle that can be drawn on the screen.

To create an drawable:

var newDrawable = new Drawable(x, y, width, height);

The x and y coordinates of an drawable are relative to the top-left corner of itself.

To assign an drawables color:

newDrawable.color = myColor;

To draw an drawable:

function Update(){
    DrawDrawable(newDrawable);
}


Color

A color is a class representing... a color.

To create a color:

var newColor = new Color(r, g, b);

Alpha values are not yet supported. :(

To assign an objects color:

newObject.color = newColor;

To assign the color of the background:

SetBackground(newColor);

SetBackground() does not have to be used in the update function to be drawn every frame.


Input

All the functions used to obtain user input

To get keyboard input:

if(isPressing('z'))
{
    Log("Z key is pressed");
}

Currently, isPressing() only supports alpha-numeric characters.
